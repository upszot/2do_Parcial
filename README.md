# 2do_Parcial

## Generador de Enunciados para el 2do parcial de Arquitectura y Sistemas Operativos

 Clonar el repositorio, ingresar en la carpeta del mismo y ejecutar:
```sh
./Genera_Ejercicios.sh
```
> Ejecutar el siguiente script para generar la estructura de directorios y el Enunciado del examen.  

- Temas a Evaluar:
  - LVM
  - Bash-Script
  - Docker
  - Ansible
